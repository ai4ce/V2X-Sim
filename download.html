Download

In this page, we provide download links to the datasets. Instructions on setting up and using the data is avaliable on the tutorial page.

Please send us an access request with your affiliation and role, and we will grant the access.

The complete directory tree structure of V2X-Sim 2.0 are listed below. However, you can download any sensors of your choice if you do not need the full dataset. For all subdirectories under /V2X-Sim/sweeps, folders with the name id_0 contain data collected from the roadside unit, while folders with the name id_1 to id_5 are data collected from the vehicles.
